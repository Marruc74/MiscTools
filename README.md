# MiscTools

A collection of small, self-contained tools. Each lives in its own folder and runs without a build step.

## Tools

| Tool | Description | How to run |
|------|-------------|------------|
| [mp3-visualizer](./mp3-visualizer) | Turn an MP3 (or WAV/OGG/M4A) into a music-visualizer video, entirely in the browser. | Open `mp3-visualizer/index.html` |
| [animation-prompt-builder](./animation-prompt-builder) | Fill a reusable template to build AI-animation story prompts in any of 20 structures (+ Master Generator). | Open `animation-prompt-builder/index.html` |

---

Each tool's own folder has a README with details.
